Riemann Heuristic Preprint v13.6
---------------------------------
This package contains LaTeX, data, Python, and figures for the v13.6 heuristic record.

Files:
- riemann_v13_6.tex : LaTeX source (compile with pdflatex)
- analysis_v13_6.py : Python reproducibility script
- data_v13_6.csv : Input data
- results_v13_6.csv : Regression results
- figure3.png : Comparative log-log regression figure
